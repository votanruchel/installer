# Installer
Instalador de sistemas PHP, automatizado

# Como usar:
1- Exporte seu arquivo de configuracao .sql e coloque na pasta raiz do programa com o nome "config.sql";
2- Configure seu arquivo base "base.php" como preferir, inserindo as sintaxes {exemplo}, e substituindo-as pelas valores no arquivo "process.php".

Adapte como voce preferir o instalador.
